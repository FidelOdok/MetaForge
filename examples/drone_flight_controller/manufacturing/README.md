# Manufacturing Output Files

This directory will contain manufacturing output files generated by MetaForge, including:

- **Gerber files** (RS-274X format): copper layers, solder mask, silkscreen, drill files
- **Pick-and-place file** (CSV): component X/Y coordinates, rotation, side
- **BOM for assembly** (CSV): consolidated BOM with manufacturer part numbers and references
- **Assembly drawings** (PDF): board outline with component placement overlay
- **Fabrication notes** (PDF): PCB specifications (layer count, thickness, surface finish, impedance)

These files are generated after the PCB layout is finalized and reviewed.
