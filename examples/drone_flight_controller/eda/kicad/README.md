# KiCad Project Files

This directory will contain the KiCad EDA project files for the DroneFC flight controller:

- **dronefc.kicad_pro** - KiCad project file
- **dronefc.kicad_sch** - Schematic (hierarchical sheets for power, MCU, sensors, connectors)
- **dronefc.kicad_pcb** - PCB layout (4-layer, 36x36mm)
- **dronefc.kicad_dru** - Design rules (trace widths, clearances, via sizes)
- **fp-lib-table** - Footprint library references
- **sym-lib-table** - Symbol library references

KiCad project files are generated by MetaForge's Electronics Agent after schematic capture and PCB layout phases are complete. In Phase 1, MetaForge performs read-only operations (ERC/DRC checks, BOM export, Gerber export) on existing KiCad files.
